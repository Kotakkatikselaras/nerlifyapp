Mengenal Perbedaan Server proxy Dan proxy terbalik

Baik server proxy dan proxy terbalik bertindak sebagai buffer antara internet dan komputer di belakangnya, tetapi dengan dua cara yang berbeda. Ingatlah bahwa server proxy bertindak atas nama klien atau jaringan klien untuk memproses lalu lintas ke dan dari server web. Proksi terbalik, seperti yang Anda bayangkan, adalah kebalikannya.

Proxy terbalik berdiri di antara kumpulan server web dan internet, dan menangani lalu lintas atas nama server tersebut. Dengan cara ini, sebuah organisasi dengan banyak server berbeda — misalnya, situs web besar dengan banyak jenis produk dan layanan — dapat menampung semua servernya di belakang satu kehadiran internet yang menghadap publik. Sama seperti proxy maju memastikan bahwa tidak ada server yang dapat secara langsung menghubungi klien di belakangnya, proxy terbalik melindungi servernya dari lalu lintas klien.

Perbedaan antara web proxy dan reverse proxy tidak kentara, tetapi signifikan. Singkatnya: proxy duduk di depan klien, dan proxy terbalik duduk di depan server .


Proses pengaturan Proxy

Anda dapat mengatur proxy dari dalam setting proxy komputer Anda, seperti yang terlihat sebelumnya di artikel ini. Anda juga dapat mengatur proxy langsung di dalam browser, meskipun dalam banyak kasus, Anda hanya akan memulai proses di sana dan menyelesaikannya dalam pengaturan OS Anda.

Singkatnya, Anda harus memasukkan alamat proxy Anda dan informasi lainnya di komputer atau pengaturan proxy browser Anda. Untuk penjelasan terperinci, lihat panduan penyiapan proxy langkah demi langkah kami untuk mengonfigurasi pengaturan proxy Anda di Windows dan macOS serta di semua browser terkemuka.

Jika Anda tidak ingin repot menyiapkan proxy, Anda dapat terhubung ke salah satu dari banyak proxy web yang tersedia secara online.
